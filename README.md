## Customization Tips

1. **For the typing SVG animations**:
   - You can add more lines by adding more entries to the `lines=` parameter
   - Adjust the `pause=` value (in milliseconds) to control typing speed
   - Change colors using hex codes in the `color=` parameter

2. **For repository cards**:
   - Replace `your-repo-1` and `your-repo-2` with your actual repository names
   - Add more repository cards as needed

3. **For the animated GIFs**:
   - You can replace the animation URLs with others that better match your style
   - Adjust width and height parameters to fit your layout

4. **For social links**:
   - Update the URLs with your actual social media profiles
   - Add or remove badges as needed
